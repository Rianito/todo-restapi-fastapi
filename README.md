## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) or [pipenv](https://pipenv.pypa.io/en/stable/) to install.

```bash
pip install -r requirements.txt
```
or
```bash
pipenv install
```

## Usage

Run the main.py file.

```bash
python main.py
```
